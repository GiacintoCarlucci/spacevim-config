```
- copy the init.toml file to ~/.SpaceVim.d folder

- open vim, it will install dependencies

- if vim-import-js build failes, install import-js using npm:
    sudo npm install -g import-js --allow-root --unsafe-perm=true
  then open vim with sudo:
    sudo -E vim
  then type in vim:
    :SPUpdate
  then quit vim
```
